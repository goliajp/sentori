## BI-five answers per module

Per [[project-v22-lens]] — flipped 模块欠 5 项答案。

**Privacy**

1. **dim** — 一行 per `(release × fieldPath × patternKind)` 的最近 finding。
2. **measure** — privacy score 0-100(server-computed risk 综合)+ findings count + top fields by frequency + risk tier(low/med/high)。
3. **drill** — finding row → sample(server-side seen-at + redacted sample bytes)。Top fields → 没 drill;它 IS 答案。
4. **compare** — n/a。score 是 absolute 而非 vs baseline。Time-window 切换 server-side 已有,UI defer。
5. **cache** — server cron 15 min `privacy_lab` 扫,React Query 60 s staleTime 在客户端。
