# Sentori v2.34 — development plan

Status: **in progress 2026-06-10 — preference center API.** Fifteenth version of Push Phase 2. Per-(project, user, category) opt-out flag. dispatch path silently skips sends to opted-out users; API surface lets the host wire a "do-not-disturb" toggle for a category.

## NEVER rule

- Zero wire break.
- Category-free sends bypass the check.
- No hosted end-user UI — API + table only (per the original design plan).
- Per-project isolation; user X in project A is independent of user X in project B.
