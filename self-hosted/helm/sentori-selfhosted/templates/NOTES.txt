Sentori self-hosted installed.

1. Wait for rollout:
   kubectl rollout status deployment/{{ .Release.Name }}-server

2. Reach the dashboard:
{{- if .Values.ingress.enabled }}
   {{- if .Values.ingress.tls.enabled }}
   open https://{{ .Values.ingress.host }}
   {{- else }}
   open http://{{ .Values.ingress.host }}
   {{- end }}
{{- else }}
   kubectl port-forward svc/{{ .Release.Name }}-server {{ .Values.server.service.port }}:{{ .Values.server.service.port }}
   open http://localhost:{{ .Values.server.service.port }}/healthz
{{- end }}

3. {{ if .Values.server.bootstrap.ownerEmail }}First owner already provisioned: {{ .Values.server.bootstrap.ownerEmail }}.
{{- else }}No bootstrap owner set — sign up via the dashboard / set `server.bootstrap.ownerEmail` to seed one.
{{- end }}
